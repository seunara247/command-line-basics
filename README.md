#command line Basics 

This repository demonstrates basic linux command usage.

##commands used:

-'mkdir'
*create directories.*

-'cd'
*change directory.*

-'touch'
*create empty file.*

-'cp'
*copy and rename file.*

-'rm'
*Delete a file.*

-'vim'
*Edit a text file to write a paragraph about my motivation for learning nodes.js.*
